# au-ts-jspm-skeleton
An Aurelia skeleton project using Typescript and JSPM, with Karma, Mocha and Chai for testing.

## Prerequesites
Ensure you hae [node]() installed.

## Restore Dependencies
```
npm run restore
```

## Build Output

### Producion
```
npm run build
```

### Development
```
npm run build-dev
```

## Serve Locally and Watch for Changes
```
npm run server
```

## Run Tests
```
karma start
```
